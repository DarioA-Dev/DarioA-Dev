## ✨ La Misión de El Nexo

¡Bienvenido\! 👋 Este repositorio aloja el código fuente de **El Nexo**, tu centro de control y portal de acceso rápido.

**¿Cansado de buscar los mismos instaladores, navegadores y herramientas de rendimiento en cada PC que usas?**

El Nexo soluciona eso: Es una página web estática que centraliza todos tus enlaces de descarga esenciales. Simplemente haz un clic y listo.

## 🛠️ Estructura del Proyecto

Este proyecto está construido con la tríada clásica de desarrollo web:

  * **HTML5:** Estructura semántica del contenido
  * **CSS4:** Estilos limpios, Flexbox, CSS Grid y el efecto *fade* en los botones
  * **JavaScript (Opcional):** Listo para futuras funciones interactivas, muy básico, futura implementación

### 📁 Carpetas Principales

Es un proyecto antiguo mío y todavía lo estoy arreglando, hay varios fallos en las rutas en cuanto a las nomenclaturas, pero esto es un poco el arbol de carpetas:

| Carpeta | Contenido | Notas |
| :--- | :--- | :--- |
| **`/src/`** | Archivos HTML de los apartados (navegadores, juegos, etc.). | Contiene las páginas secundarias. |
| **`/style/`** | Archivos CSS (ej. `main.css`). | Define toda la estética, tipografía ('Orbitron', 'Montserrat'), y el *layout*. |
| **`/img/`** | Iconos y logos del proyecto (ej. `nexo.png`). | |
