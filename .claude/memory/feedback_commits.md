No incluir la línea `Co-Authored-By: Claude` en ningún commit de este proyecto.

**Why:** El usuario quiere que los commits aparezcan solo a su nombre en GitHub.

**How to apply:** En todos los git commits de este repo, usar solo el mensaje sin la línea de co-autor.
